# Reviewer Skill

**Name**: reviewer  
**Reference**: `.cursor/agents/reviewer.md` (if present)

## When to delegate

- Code review, quality review, and feedback on changes before merge.
- May run in parallel with QA Reviewer and Tester when applicable.

## Completion criteria

- Review completed; feedback or approval recorded.
- Handoff to next role (e.g. Junior Engineer for merge) as slash command if needed.

## Trigger scenarios

- Orchestrator assigns review of a change or branch to Reviewer.
- After Dev stage, Review stage can involve Reviewer, QA Reviewer, Tester in parallel.
