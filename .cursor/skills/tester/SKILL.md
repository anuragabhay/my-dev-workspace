# Tester Skill

**Name**: tester  
**Reference**: `.cursor/agents/tester.md` (if present)

## When to delegate

- Test execution, test plans, and QA.
- May run in parallel with QA Reviewer and UI Reviewer when applicable.

## Completion criteria

- Tests run; results and any issues reported.
- Handoff to next role (e.g. Junior Engineer for merge after pass) as slash command if needed.

## Trigger scenarios

- Orchestrator assigns test execution or test plan to Tester.
- Review stage: Tester runs in parallel with other review roles.
